# weex-ShareSdk
weex-ShareSdk是一个weex分享插件， 可以通过weexpack快速集成，可以丰富weex功能

支持的weexpack版本： >= 0.2.0
支持的WeexSDK版本： >= 0.10.0

# 功能

# 快速使用
- 通过weexpack初始化一个测试工程 weextest
   ```
   weexpack create weextest
   ```
- 添加ios平台
  ```
  weexpack platform add ios
  ```
- 添加android平台
  ```
  weexpack platform add android
  ```
- 添加插件
  ```
  weexpack plugin add weex-ShareSdk
  ```

# 已有工程集成
## iOS集成插件WeexShareSdk
- 命令行集成
  ```
  weexpack plugin add weex-ShareSdk
  ```
- 手动集成
  在podfile 中添加
  ```
  pod 'WeexShareSdk'
  ```
